## rom-tools screenscraper search

Search for games by name

### Synopsis

Searches for games by name, returns up to 30 results sorted by probability

```
rom-tools screenscraper search <query> [flags]
```

### Options

```
  -h, --help            help for search
  -s, --system string   System ID to filter results
```
