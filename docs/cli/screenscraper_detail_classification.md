## screenscraper detail classification

Get detailed classification information

### Synopsis

Retrieves detailed information about a specific classification/rating

```
screenscraper detail classification [flags]
```

### Examples

```
  screenscraper detail classification --id=1
  screenscraper detail classification --name=PEGI
```

### Options

```
  -h, --help          help for classification
      --id int        Classification ID
      --name string   Classification short name
```
